OCR Panel Example

This is a re-creation of the (Orange County Register) OCR Panel using Google Maps API V3. The original was built with Google Maps API V2 and integrated custom traffic, restaurant inspection, and real estate data in addition to the crime data. Each panel also had an associated datatable that could be shown or hidden.

Use the "More..." button to see crime markers.
